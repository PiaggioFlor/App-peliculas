# App-peliculas
---

## Funcionalidades

- Búsqueda de películas
- Visualización de detalle
- Agregar a favoritos con preferencias
- Eliminación de favoritos
- Historial de visualización

---

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- API OMDb

---

## API utilizada

Se utilizó la API de TMDB para obtener información de películas en formato JSON.

---

## Almacenamiento

Los datos de favoritos e historial se almacenan en el navegador utilizando localStorage.

---

## Cómo usar

1. Ingresar una película en el buscador  
2. Seleccionar un resultado  
3. Visualizar el detalle  
4. Agregar a favoritos si se desea
